Place your skin image(s) in this folder.

For the 'Türk Bayrağı' skin added as a development example, the repo includes an SVG at:

    assets/turkey.svg

Reload the page — the lobby/store will show the skin thumbnail and you can equip it.

If you want to replace it with a different image, save your file to `assets/` and update the `image` field in `client.js` dev store or on the server-side store data.
